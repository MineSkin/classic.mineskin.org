About

MineSkin.org allows you to generate skin texture data signed by Mojang.
These can be used on ingame skull blocks or to change a player's skin using some packet magic :).

This website makes custom skins a lot more convenient and less time consuming, since it uses multiple accounts donated by multiple awesome people <3. This way, you don't have to use your own account to generate textures.
If you'd like to make things even faster, please contact inventivetalent to donate a Minecraft account :) (more details)

There's also a Java client for the generator API, if you need to generate a lot of different textures.

If you have suggestions or find a problem, please post it on the GitHub issue tracker or tweet @MineSkinOrg or @Inventivtalent.
